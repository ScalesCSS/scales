Horizontal List Object for Scales

This page includes the default horizontal list object styles

Horizontal List usage

Horizontal List should be used whenever you need to make an ordered or unordered list horizontal instead of vertical. You can extend it with '--delimited' to semantically mark up a character delimited list for things like tags, etc. You can also center, right align, or fill the available space.
